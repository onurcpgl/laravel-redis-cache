# Laravel Redis Cache

Bu proje, Laravel uygulamamda Redis üzerinden cache kullanarak veri çekme ve veritabanından veri çekme işlemlerini göstermektedir.
3000 veri üzerinden bu işlemler gerçekleştirilmiştir.
